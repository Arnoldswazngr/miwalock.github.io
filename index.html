L2) Guest Cards do not work, but staff cards are able to access the door.

This may be due to the set time in the lock being wrong. Please refer to the reprogramming your lock guide in L1.

You can also test this by creating a guest card for the room, using it on the door to have it light up red (when a door lights red, it writes an error code into the card). Bring back the card to the front desk and select Read Guest Card. Look for an error type in the description. If it is out of staying period, then it has the wrong time and reprogramming the lock/retesting the card should correct the issue.
